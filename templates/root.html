Movie Recommendation System API

This is a personal project involving ETL, FastApi and Machine Learning. If you want to try these functions with differents inputs, you need to change the default value in the navigation bar of your browser. Example, change "/cantidad_filmaciones_mes/enero" with "/cantidad_filmaciones_mes/febrero" or go to the Swagger User Interface..

  • Calculate the number of movies given a month in Spanish. Try it with enero
  • Calculate the number of movies given a day of the week in Spanish. Try it with viernes.
  • Obtain the popularity of a movie by entering the title. Try it with "12 Angry Men".
  • Obtain the number of votes of a movie by entering the title. Try it with"Toy Story".
  • Obtain the total budget, total revenue, and return of the movies given an actor."Johnny Depp".
  • Obtain the total budget, total revenue, and return of the movies given a director."Michael Bay".
  • Get movie recommendations. Try it with Cars.
